# Contributing

This repository is public for transparency and documentation.

Please do not push directly to `main`. Open a pull request and wait for maintainer approval.

## Pull Request Checklist

- Explain what changed.
- Keep permissions narrow.
- Do not add broad host access.
- Do not add remote code execution.
- Do not add analytics, tracking, or advertising code.
- Update `docs/SERVICE_WORKER.md` when the service worker contract changes.
